## 简介:

一款基于 rollup 与 typescript 的轻量级脚手架，专注于 JavaScript 库的开发，能够满足开发库时所需要的各项功能；

包含功能:

- rollup | tree-shaking
- typescript
- hot-reload
- example / debug
- sass

## 开发姿势:

- `npm run dev`: 开发环境
- `npm run publish`: 打包编译
